# Godwit Packages

Package manager manifests for [Godwit](https://godwit.io) tools.

## Godwit Sync

Plan-first S3 object storage migration and sync CLI.

### Homebrew (macOS, Linux)

```bash
brew tap godwitio/packages https://github.com/godwitio/packages
brew install godwit-sync
```

### Scoop (Windows)

```bash
scoop bucket add godwit https://github.com/godwitio/packages
scoop install godwit-sync
```

### Chocolatey (Windows)

```bash
choco install godwit-sync
```

### winget (Windows)

```bash
winget install GodwitIO.GodwitSync
```

### Direct download

Pre-built binaries and Docker images are available at [godwit.io/downloads](https://godwit.io/downloads).
